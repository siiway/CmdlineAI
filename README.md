# CmdlineAI

一个简单的命令行 AI 客户端

### 功能 / 支持

目前的功能:

- [x] 简单对话
- [ ] 会话保存
- [ ] 数据同步
- [ ] 流式输出

目前支持的渠道:

- [x] Cloudflare AI
- [ ] OpenAI
- [ ] Others

### 开发配置

开发环境的配置和发布隔离, 需手动创建 `config.dev.json` 并写入以下内容:

```json
{
    "init": true
}
```

来初始化配置
